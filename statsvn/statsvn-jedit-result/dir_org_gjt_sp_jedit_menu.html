daleanson 2010-01-01 20:33 Rev.: 16805

Fix for 2924261, BufferLocal ignores request to change encoding. The problem isn't that BufferLocal ignores the request, the problem is that the encoding property is never set on the buffer before it is closed, so BufferLocal is never notified that the user wanted it changed. This change sets the encoding prior to closing the buffer, so the proper EditBus message is generated, so BufferLocal is notified.
