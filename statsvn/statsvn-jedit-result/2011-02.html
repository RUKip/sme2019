Entire word search rewritten :
now the search is done as usual, then before returning the result it is checked to be a whole word, if not, the next result is searched (and checked).

A whole word has the following conditions :

start of word :
beginning of the line
starts with a symbol or space
char before the first char is a symbol or space

end of word:
end of line
ends with a symbol or space
char next to the last char is a symbol or space
