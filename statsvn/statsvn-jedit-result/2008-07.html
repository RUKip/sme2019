k_satoda 2008-07-14 20:14 #20 Rev.: 13049

Made "Close (local)" as the default action to close a buffer because it
is more natural action than current "Close" which affects to inactive
editpanes or views.

Here is the summary of changes.

From:
action "editpane-close-buffer" label "Close (local)"
action "close-buffer" label "Close"
To:
action "close-buffer" label "Close"
action "global-close-buffer" label "Close (global)"

And also, an API name is changed to match with the action name;
From
jEdit.removeFromBufferSet(EditPane, Buffer)
To:
jEdit.closeBuffer(EditPane, Buffer)
// This is an overload for jEdit.closeBuffer(View, Buffer)
