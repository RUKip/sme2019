Made localized .props file to be in UTF-8 so that they can be edited
directly without escaping non ISO-8859-1 characters.

Existing files were migrated as follows:
- Convert all existing escapes into the original characters
represented in UTF-8.
#!/bin/sh
for PROPS in *.props; do
native2ascii -encoding ISO-8859-1 $PROPS >$PROPS.ascii
native2ascii -reverse -encoding UTF-8 $PROPS.ascii >$PROPS
done
- Replace trailing spaces in values into "\u0020" not to be stripped
accidentally in the future.
- Remove all other trailing spaces in comments or empty lines.
- Set ":encoding=UTF-8:".
