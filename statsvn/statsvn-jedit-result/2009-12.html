shlomy 2009-12-02 22:09 #3 Rev.: 16591

Fixed bug #2907139: Upgrading from 4.2final to 4.3pre18 causes freeze at startup
The freeze was caused by an NPE while restoring the view's split configuration, because the buffersets feature changed both the format of the split config string and also created a new section in the perspective file for the specification of the buffers to open on startup. The new section does not exist in old perspective files, so the buffers are not opened automatically before reading the split config, and restoring the split config expected them to be open (with the buffersets feature).
