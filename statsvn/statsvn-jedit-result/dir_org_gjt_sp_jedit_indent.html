Another tweak to deep indent: the current code could get confused when the code had lots of intertwined parenthesis (for example, several embedded function calls inside a conditional spanning several lines).

To avoid this confusion, keep data to match opening and closing parenthesis when parsing the lines. This way the code can know for sure what is the unmatched parenthesis and its position, and then do the proper alignment.
